130万円を超えると自分で国保・年金を払う必要が出て**年間約30万円の出費増**。逆に130万円未満なら扶養のメリットをフルに受けられます。

**社会保険の130万が先に来るので、130万円以内なら所得税も完全にゼロ**になります。

### 社会保険の130万円の壁（最重要）

親の健康保険の被扶養者でいるための条件です。**経費を引く前の売上が130万円未満**であることが必要です。

「**家内労働者等の必要経費の特例**」を使うと、実際の経費が少なくても65万円を経費にできます。基礎控除95万と合わせて**160万円まで非課税**なので、130万円なら余裕でクリアです。

<!-- 確定申告の注意 -->
<div style="background: #fefce8; border: 2px solid #fde047; border-radius: 12px; padding: 20px; margin: 16px 0;">
  <p style="font-size: 15px; font-weight: 900; color: #713f12; margin: 0 0 8px 0;">ただし確定申告は必要！</p>
  <p style="font-size: 13px; color: #78350f; margin: 0 0 8px 0;">特例は自動で適用されるわけではありません。確定申告で**特例にチェックを入れて初めて65万円の経費が認められます**。</p>
  <p style="font-size: 13px; color: #78350f; margin: 0;">チェックを忘れると特例が適用されず、収入がそのまま所得になって税金がかかってしまいます。レシート集めは不要ですが、確定申告だけは忘れずに！</p>
</div>

## 家内労働者の特例って何？

<!-- 特例の説明カード -->
<div style="background: #fefce8; border: 2px solid #fde047; border-radius: 12px; padding: 20px; margin: 20px 0;">
  <p style="font-size: 15px; font-weight: 900; color: #713f12; margin: 0 0 8px 0;">家内労働者等の必要経費の特例</p>
  <p style="font-size: 13px; color: #78350f; margin: 0 0 12px 0;">「特定の1社から継続的に業務委託を受けている人」が使える制度。実際の経費が少なくても<strong>65万円まで経費として認められる</strong>。</p>
  <div style="display: grid; grid-template-columns: 1fr 1fr; gap: 8px;">
    <div style="background: white; border-radius: 8px; padding: 10px; text-align: center;">
      <p style="font-size: 18px; margin: 0;">✅</p>
      <p style="font-size: 11px; color: #166534; font-weight: bold; margin: 4px 0 0 0;">Uber Eatsのみ</p>
      <p style="font-size: 10px; color: #6b7280; margin: 2px 0 0 0;">1社契約なので適用OK</p>
    </div>
    <div style="background: white; border-radius: 8px; padding: 10px; text-align: center;">
      <p style="font-size: 18px; margin: 0;">⚠️</p>
      <p style="font-size: 11px; color: #b45309; font-weight: bold; margin: 4px 0 0 0;">Uber＋出前館など</p>
      <p style="font-size: 10px; color: #6b7280; margin: 2px 0 0 0;">掛け持ちは適用NGの可能性</p>
    </div>
  </div>
</div>

バイト（給与収入）がある場合は、給与所得控除との合計で65万円が上限です。例えばバイトで年40万の給与がある場合、特例で使える経費は65万−40万＝**25万円に減ります**。Uber Eatsだけで稼ぐのが最もシンプルです。

家内労働者の特例を使う場合は、確定申告時に**「必要経費の額の計算書」を添付**する必要があります。
